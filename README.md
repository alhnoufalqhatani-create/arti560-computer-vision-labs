# ARTI 560 Computer Vision Laboratory 

## Overview
This repository contains code, datasets, and exercises for various **computer vision tasks** using **TensorFlow** and **Keras**. The labs are designed to teach practical concepts in computer vision, including image classification, object detection, segmentation, and more.

## Objectives
- Learn fundamental and advanced computer vision techniques.
- Work with pretrained models and transfer learning.
- Explore model architectures, performance, and optimization.
- Apply computer vision methods to real-world datasets.

## Requirements
- Python 3.10+  
- TensorFlow, Keras, and common data science packages  

## Installation Using Conda

1. **Clone the repository**
```
bash
git clone <repository_url>
cd <repository_folder>
```

2. Create a Conda environment
```
conda create -n cv_lab python=3.10
```
3. Activate the environment
```
conda activate cv_lab
```

4. Install required packages

```
pip install -r requirements.txt
```
